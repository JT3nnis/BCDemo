[Installation]
Visual Studio 2019
Make sure to install from the tools and features section
- SQL Server
- Entity Framework 6

SSMS (SQL Server Management Studio 18)
SQL Server 2019 (Developer)

[Setup]
You will need to have your SQL Server running locally.
Clone the Repo
Open SSMS and run the BuildersCaptail.sql SQL Script
Edit the App.config connection string in the BuildersCapitalDataAccess Project (Replace the datasource with your SQL Datasource)
Edit the Web.config connection string in the BuildersCapital Project (Replace the datasource with your SQL Datasource)
